== Description ==

The Ribbon Manager plugin give the ability to add a side ribbon on the blog without touching the template file or the css.
Please note that the plugin needs a theme with <code>wp_head();</code> and  <code>wp_footer();</code> enabled.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `fcc_ribbon.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go in `Option->Ribbon Manager` and configure it
